    双天线定向配置

    默认航向

      B92（Rover移动站模式），B94，B96三款板卡支持双天线定向（Heading）功能，此航向信息完全由卫星导航系统计算得出，相比地磁传感技术，避免了大体积金属，地磁异常等引起的误差和干扰，也无需校准，可靠性更高，使用更方便。
      以B92为例，板卡默认航向示意图如下：

      B92 ANT1定位天线与ANT2定向天线之间的基线向量，与真北方向逆时针的夹角。如上图所示假设上北下南，那此时航向角为0度，飞控此时的航向角也为0度，不需额外调整参数。

    基线长度

    两个天线的水平直线距离称为基线长度，基线长度过小会影响航向角精度，因此在条件允许的情况下尽量使基线长度大于40CM。

    串口助手配置

      请按照上面六轴多旋翼图示安装天线1和天线2以及飞控，确认无误。
      以串口助手sscom为例，按照下图中所示的顺序操作即可。使用其它串口助手操作类似。

      至上图中第5步则配置成功。只需配置一次即可，板卡自动保存。
      配置命令如下： 

    CONFIG HEADING OFFSET -90 0
saveconfig

    注意

    1. 所有字符为纯英文字符，勿使用中文字符。
    2. 两行命令后面都有回车换行（不可见），建议使用串口助手自带的加回车换行功能。
